# Security Policy

## Supported Versions

Security updates are handled on the default branch. Use the latest release tag when downloading binaries.

## Reporting a Vulnerability

Please report security issues privately through GitHub's private vulnerability reporting when available, or contact the repository owner directly.

Do not include private registry data, personal data dumps, access tokens, or non-public CZ.NIC information in public issues.

## Data Handling

`czdomains` only uses public best-effort sources and public CZ.NIC RDAP responses. It does not bypass hidden or redacted contact data.
